## 打包

如果您要自己打包，请先在 [汉仪字库 - 汉仪文黑](https://www.hanyi.com.cn/productdetail?id=987) 获取一份个人非商业授权，之后下载字体文件并拷贝至 `system/fonts` 文件夹下。

```bash
system/fonts
├── HYWenHei-35W.ttf
├── HYWenHei-45W.ttf
├── HYWenHei-55W.ttf
├── HYWenHei-65W.ttf
├── HYWenHei-75W.ttf
└── HYWenHei-85W.ttf
```
